  [1X6.1-1 IsMatrixInRREF[101X
  
  [33X[1;0Y[29X[2XIsMatrixInRREF[102X( [3Xmat[103X ) [32X function[133X
  [6XReturns:[106X  [33X[0;10Ybool[133X
  
  [33X[0;0YChecks  whether or not a matrix mat is in RREF where RREF means 'Reduced Row
  Echelon Form'. A matrix is in RREF if it meets the following criteria:[133X
  
  [30X    [33X[0;6YThe first nonzero entry of a nonzero row is 1 (called 'leading 1').[133X
  
  [30X    [33X[0;6YWhen  looking  at  two nonzero rows, than the leading one of the lower
        rows is further right than the other leading one.[133X
  
  [30X    [33X[0;6YThe rows containing only zeros are below the nonzero rows.[133X
  
  [30X    [33X[0;6YEvery  column  that  contains a leading one must have zeros everywhere
        else in that column.[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27XM := RandomMat( 3, 3 );;[127X[104X
    [4X[25Xgap>[125X [27XDisplay( M );[127X[104X
    [4X[28X[ [   1,   0,  -1 ],[128X[104X
    [4X[28X  [  -1,  -1,   1 ],[128X[104X
    [4X[28X  [  -1,   1,  -2 ] ][128X[104X
    [4X[25Xgap>[125X [27XIsMatrixInRREF( M );[127X[104X
    [4X[28Xfalse[128X[104X
    [4X[25Xgap>[125X [27XN := IdentityMat( 3 );;[127X[104X
    [4X[25Xgap>[125X [27XDisplay( N );[127X[104X
    [4X[28X[ [  1,  0,  0 ][128X[104X
    [4X[28X  [  0,  1,  0 ],[128X[104X
    [4X[28X  [  0,  0,  1 ] ][128X[104X
    [4X[25Xgap>[125X [27XIsMatrixInRREF( N );[127X[104X
    [4X[28Xtrue[128X[104X
    [4X[25Xgap>[125X [27XL := [ [ 1, 0, 3 ], [ 0, 1, 5 ] ];;[127X[104X
    [4X[25Xgap>[125X [27XDisplay( L );[127X[104X
